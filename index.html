SQL used for this analysis

Key queries — see GitHub for all 7

SELECT
  "Provider Org Name",
  SUM(CAST("Total All" AS INTEGER))
    AS total_waiting
FROM waiting_times
WHERE "RTT Part Type" = 'Part_2'
GROUP BY "Provider Org Name"
ORDER BY total_waiting DESC
LIMIT 10;
